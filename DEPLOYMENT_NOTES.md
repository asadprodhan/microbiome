# Deployment Notes

This package is a static GitHub Pages website. It does not require Jekyll theme setup.

## How to deploy

1. Delete the old generated files from the repository if needed.
2. Upload all files and folders from this ZIP to the root of your repository.
3. Keep the repository public.
4. Go to Settings → Pages.
5. Select Deploy from a branch.
6. Select branch: main.
7. Select folder: / root.
8. Save.

Your site will be available at:

https://asadprodhan.github.io/BCRTI_Micro-Credential_Microbiome_Analysis_v2/

or the URL matching your final repository name.
